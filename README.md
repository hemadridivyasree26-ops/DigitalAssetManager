# ✨ Digital Asset Management System

A simple web-based Digital Asset Management System mini project that helps users upload, store, view, and manage digital files like images and documents.

## 📌 About The Project

The Digital Asset Management System allows users to manage digital assets in one place. Users can upload files, store asset details in a database, view uploaded files, and delete unwanted assets.

This project is built as a mini project using PHP and MySQL.

## 🚀 Features

- 📤 Upload digital assets
- 🗂️ Store asset details
- 🖼️ View uploaded files
- 🗑️ Delete assets
- 💾 MySQL database integration
- 🎨 Simple and cute user interface
- 📱 User-friendly design

## 🛠️ Technologies Used

- HTML
- CSS
- PHP
- MySQL
- XAMPP
